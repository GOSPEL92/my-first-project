# Hospital Performance Dashboard

## Overview
This project contains a Power BI dashboard for analyzing hospital performance.  
Currently, the focus is on the **Complications Performance Page**, which highlights:
- Average Complication Score
- Best/Worst Facility Scores
- Compared to National Benchmarks
- Trends over time

Future updates will include Engagement, TPS, and Clinical Outcomes pages.
